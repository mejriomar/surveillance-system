# Smart Watching System

This project aims to build a full and smart watching system, able to watch a place in real time, spot odd things, and send warnings through a simple user interface. The system brings together several parts to keep safety at its best.

## Project overview

The smart watching system is made of several linked modules to give full protection:
- **Watching camera**: Real-time video watching.
- **Shock detection**: Spotting hits or strange shakes.
- **Access control**: Handling physical and digital entry.
- **Gas, temperature, and smoke detection**: Watching environmental states.
- **Smart warning system**: Real-time warnings through many channels.

---

## System parts

### 1. **Watching camera**
- Uses IP cameras or webcams for video watching.
- Features: motion spotting and video recording.

### 2. **Shock detection**
- Vibration sensors or accelerometers to spot hits.
- Sends a warning in case of a shock or break-in attempt.
- Works with the central system for a quick response.

### 3. **Access control**
- A system for handling physical entry (badges, electronic keys).
- Digital checking (fingerprint, face finding).
- Logs of ins and outs.

### 4. **Gas, temperature, and smoke detection**
- Sensors for gas (CO, CO2, and more), temperature, and smoke.
- Real-time watching of environmental states.
- Sends a warning when levels are dangerous or smoke is found.

### 5. **Smart warning system**
- Real-time notices by email, SMS, or mobile app.
- Custom warnings based on how serious the event is.

## Graphical interfaces

### 1. **Main dashboard**
- Shows live video feeds.
- Displays sensor data (temperature, gas, smoke).
- Control panel to turn modules on or off.

### 2. **Warning management view**
- List of warnings with details (time, event type, place).
- Options to set notifications (channels, thresholds).

### 3. **Access control view**
- Manages users and permissions.
- Logs access and unauthorized attempts.

## Use cases

### Use case 1: Motion detection and warning
- A person enters a watched area.
- The camera spots the motion and sends a warning through the notice system.

### Use case 2: Shock detection
- A shock is found near a door or window.
- The system sends a quick warning and records the matching video.

### Use case 3: Smoke detection
- A sensor finds smoke in a watched area.
- The system triggers a warning and starts emergency steps.

### Use case 4: Access control
- A user tries to enter a secured area.
- The system checks permissions and logs the entry.
